# Phase 0 Approved Reuse List

Phase 0 does not approve copying upstream source files into Vyu.

Approved activities:

- Clone upstream repositories into `upstreams/` as read-only reference material.
- Record exact commit SHAs, licence files, licence hashes, and dependency manifests.
- Study architecture, connector, retrieval, and evaluation patterns.
- Create Vyu-owned interfaces and implementation plans based on reviewed patterns.

Not approved in Phase 0:

- Copying source from GPL, unlicensed, or licence-unverified repositories.
- Importing upstream packages as runtime dependencies.
- Downloading model weights, embeddings, corpora, or biomedical datasets.
- Representing upstream README licence statements as final legal clearance when a licence file is missing or conflicting.
